감성 일상 숏츠 트렌드 리포트

갱신 시각 불러오는 중…

이 페이지는 내 채널 니치(감성 일상 브이로그)에서 수집한 영상 샘플에서 읽어낸 패턴을 정리한 리서치 초안입니다. 통계적 트렌드라기보단 "이 샘플에서 뭐가 보이나"를 참고용으로 옮긴 것입니다.

이 리포트를 읽기 전에 (한계 명시):
  • 표본이 작습니다. 기본 30건, 정직 필터(니치 + 최근 90일 + 조회수 1000+) 적용 후엔 그보다 적을 수 있고, 조건을 동시 충족하는 영상이 없으면 n=0일 수 있습니다.
  • 1회 스냅샷입니다. 특정 시점 1회 수집이며 시계열 추세가 아닙니다.
  • BGM은 음원 인식(ACRCloud) 미사용 시 식별되지 않습니다. 제목 기반 추정을 하지 않습니다.
  • 추천 기획은 Claude API 미사용 시 수동 초안입니다 (자동 분석 아님).

6갱신 방법

매일 새벽 자동 갱신 — GitHub Actions가 이 리포지토리의 파이프라인을 돌려 output/*.json을 다시 뽑고, 이 페이지를 다시 렌더합니다.

수동 갱신 — GitHub 리포지토리 Actions 탭에서 Run workflow 버튼으로 즉시 다시 돌릴 수 있습니다.

자동 추천(Claude)·실제 BGM 식별(ACRCloud)을 켜려면 ANTHROPIC_API_KEY / ACR_ACCESS_KEY / ACR_ACCESS_SECRET을 GitHub Secrets에 등록하세요. 등록 시 자동 추천이 활성화됩니다.

로컬에서 이 페이지를 보려면 CORS 제한 때문에 python3 -m http.server 실행 후 http://localhost:8000/로 열어야 합니다.
